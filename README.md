## About

This project was written in the exam at the **UNIT Factory** which is the Ukrainian analogue of the French **School 42**. The coding style fully satisfies the wishes of the **Norminette**.

Interpreter **does not check for errors**, since the program always receives the correct code which using maximum **2048 bytes of memory**.

## Build

Just like all the ingenious:

```bash
$> gcc -Wall -Wextra -Werror main.c
```

Running:

```bash
$> ./a.out "++++++++++[>+++++++>++++++++++>+++>+<<<<-]>++.>+.+++++++..+++.>++.<<+++++++++++++++.>.+++.------.--------.>+.>."
```
